Instructions:

1. Unpack tarball
    
    tar -xvf hw3.tar
    
2. Navigate to the hw3 directory

    cd hw3
    
3. Compile DHClient.java

    javac DHClient.java
    
4. Compile DHServer.java
    
    javac DHServer.java
    
5. Run DHServer
    
    java DHServer 
    
    Note: If you would like to see the console output of the DHServer, run this command in its own shell program. 
    Otherwise, you can still see the output in the output.txt file
    
6. Run DHClient

    java DHClient
